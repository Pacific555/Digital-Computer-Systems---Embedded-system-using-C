# Digital-Computer-Systems---Embedded-system-using-C

## About the Project

### My Role & Contributions

- Designed the full system architecture (MCU + PC GUI)
- Implemented bare-metal firmware in C for MSP430
- Developed custom UART protocol and FSM controller
- Built flash-based file system with metadata management
- Implemented ISR-based communication and power modes
- Developed Python GUI for visualization and control
- Integrated sensors, servo, ADC, and timers

- Real-time systems
- Hardware/Software integration
